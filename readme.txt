## Description

Java pacman game with multiple map selection and game replay.
Game objective is to collect all keys on map and get to the target (chest) without being caught by ghosts.
Game is controlled by arrow keys. Replay of the game can be controlled with buttons on bottom of the screen.

## Compilation and execution

Game package can be created with `mvn package`
Game can be compiled and executed using `mvn javafx:run` command in project root directory.
